## Usage:
clone repo

```bash
git clone https://github.com/nicholasikiroma/vonage-verify_app
```

Create virtual environment
```bash
virtualenv venv
```
Activate virtual environment
```bash
source venv/bin/activate
```

Install dependencies
```bash
pip3 install -r requirements.txt
```
